# ToCSIII_TBLConverter
Convert .tbl file from/to CSV

## Usage:
Tool usable from command line or by dragin the file you want to convert to the executable.

## Examples:
Convert the file foo.tbl:
```
ToCSIII_TBLConverter.exe foo.tbl
```
